# Book Review Sentiment Analysis

A machine learning project that performs sentiment analysis on Kindle book reviews to classify them as positive or negative.

## 📊 Project Overview
- **Accuracy**: 85.25% with Multinomial Naive Bayes + TF-IDF
- **Dataset**: 12,000 Kindle reviews
- **Models**: Naive Bayes, Random Forest
- **Features**: Bag-of-Words, TF-IDF
